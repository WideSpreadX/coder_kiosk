What is HTML

HTML stands for HyperText Markup Language.

Unlike a scripting or programming language that uses scripts to perform functions, a markup language uses tags to identify content

The Web Structure

The ability to code using HTML is essential for any web professional. Acquiring this skill should be the starting point for anyone who is learning how to create content for the web.

The html Tag

Although various versions have been released over the years, HTML basics remain the same.

The structure of an HTML document has been compared with that of a sandwich. As a sandwich has two slices of bread, the HTMLL document has opening and closing HTML tags.

These tags, like the bread in a sandwich, surround everything else

The head Tag

Immediately following the opening HTML tag, you'll find the head of the document, which identified by opening and closing head tags.

The head of an HTML file contains all of the non-visual elements that help make the page work.

The body Tag

The body tag follows the head tag. All visual-structural elements are contained within the body tag.

Headings, paragraphs, lists, quotes, images, and links are just a few of the elements that can be contained within the body tag.

The HTML File

HTML files are text files, so you can use any text editor to create your first webpage. Recommended: Visual Studio Code.

All HTML files need to be saved with a .html extension, in order to be read by browsers.

To place a title on the tab describing the web page, add a =title= element to your head section.

HTML Basics

Paragraphs

To create a paragraph, simply type in the pTAG element with its opening and closing tags.

You can use the br/ tag to add a single line of text without starting a new paragraph. The br/ element is an empty HTML element. It has no end tag.

It is best practice to just make a new pTAG for every paragraph, your code will be much more organized.
